# Argomachine: Oracle-Guided Evolutionary Dynamics

Computational implementations for the paper "Evolutionary Dynamics of the ArgoMachine: 
An Oracle-Guided Computational Model of Biological Adaptation" (Kuznetsov, 2026).

## Contents
- `argo1/` — Baseline model
- `argo3/` — Probabilistic model with soft matching
- `argo8/` — Model with transposition
- `argo10/` — Oracle-guided evolution
- `argo16/` — Enhanced multi-criteria adaptation
- `data/` — Raw experimental data
- `figures/` — Reproduction scripts for all figures

## Requirements
Python 3.9+
numpy, matplotlib, scipy, pandas

## Usage
See individual README files in each subdirectory.
